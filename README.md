# CraftConnect
**Empowering Local Artisans Through Digital Commerce.**

![Dashboard Screenshot](CraftConnect/public/dashbaord.png)

## Overview
CraftConnect is a beautifully designed, modern web application that bridges the gap between traditional craftsmanship and global commerce. Born out of a mission to preserve the rich cultural heritage of Jammu & Kashmir, the platform empowers local artisans—wood carvers, shawl makers, embroidery artists, and rural women's self-help groups—by providing them a direct and transparent marketplace to sell their authentic creations.

## Key Features
- **Artisan Marketplace:** A storefront directly connecting buyers with creators, cutting out exploitative middlemen.
- **Creator Profiles:** Beautiful portfolios highlighting the stories, heritage, and products of individual artisans.
- **Education Hub:** A dedicated section featuring documentaries and articles to educate the world about traditional Kashmiri crafts.
- **Modern & Premium Design:** A breathtaking aesthetic with a custom Rust/Terracotta theme, smooth scroll, and Framer Motion micro-animations that reflect the premium nature of the products.
- **Responsive Layout:** Perfectly optimized for mobile and desktop viewing.

## Tech Stack
- **Frontend Framework:** React 19 + Vite
- **Styling:** Custom CSS with CSS Variables and CSS-in-JS
- **Animations:** Framer Motion
- **Routing:** React Router DOM v7
- **Deployment:** Vercel

## Getting Started

To run the application locally:

### Prerequisites
- Node.js installed

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Burhanali2211/Craft-Connect.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Craft-Connect/CraftConnect
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Mission
CraftConnect was born from a simple belief: the exquisite heritage of Jammu & Kashmir deserves a global stage, and the artisans who create it deserve fair, direct compensation.
